# EE6222-MachineVision
Assignment for NTU EE6222 Machine Vision, from 2023-2024 semester 1. 

Lectured by Prof. Jiang Xudong &amp; Dr. Xu Yuecong.

# Repository Overviewkv
- `Ass1_DimReduc/`: Assignment 1 - Dimensionality Reduction.
  - `submission.pdf`: submission report.
  - `instructions.pdf`: instructions for the assignment.
  - `logi.py`: python script for **logistic regression**.
  - `maha.py`: python script for **Mahalanobis distance**.
  - `data/`: iris dataset.
  - `reference/`: reference materials.
- `Ass2_HAR/`: Assignment 2 - Human Action Recognition.
  - `submission.pdf`: submission report.
  - `instructions.pdf`: instructions for the assignment.
  - `code/`: python scripts for the assignment.
    - `nn.py`: python script for **neural network**.
    - `svm.py`: python script for **support vector machine**.
    - `lstm`: python script for **LSTM**.
  - `data/`: HAR dataset.
